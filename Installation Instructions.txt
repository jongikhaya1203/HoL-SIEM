  Quick Reference

  Installation (5 Steps):

  1. Install XAMPP → Start Apache & MySQL
  2. Extract files to C:\xampp\htdocs\networkscan\
  3. Create database "network_security" in phpMyAdmin
  4. Run setup scripts:
     php setup_multitenancy.php
     php setup_authentication.php
     php create_agent_tables.php
  5. Login at http://localhost/networkscan/login.php
     Username: admin | Password: admin123

  Agent Deployment:

  1. Go to http://localhost/networkscan/agents.php
  2. Download Windows Installer (.bat file)
  3. Right-click → Run as Administrator
  4. Agent appears in dashboard within 5 minutes
