1 Setup Database

Choose one of these methods:

Method A: Using phpMyAdmin (Recommended)
  1. Open http://localhost/phpmyadmin
  2. Click on "Import" tab in the top menu
  3. Click "Choose File" and select: mailscan_install.sql
  4. Scroll down and click "Import" button
  5. Wait for success message!
Open phpMyAdmin
Method B: Using Command Line
  1. Open Command Prompt in the networkscan folder
  2. Run: php install_database.php
  3. Wait for completion message
Method C: Using MySQL Command
  1. Open Command Prompt
  2. Navigate to: cd C:\xampp\mysql\bin
  3. Run: mysql -u root -e "source C:/xampp/htdocs/networkscan/mailscan_install.sql"
2 Generate Sample Data

Create 10 sample emails with various types of sensitive information:

  1. Open Command Prompt in: C:\xampp\htdocs\networkscan
  2. Run: php generate_sample_data.php
  3. This will insert emails and scan them automatically

Tip: You can skip this step if you want to start with a clean system.

3 Access the System

Open these pages in your browser:

  • Home Page - mailscan_index.php
  • Dashboard - View flagged emails
  • Detection Rules - Manage scanning rules
  • Scan Email - Test the scanner
Open Email DLP System
