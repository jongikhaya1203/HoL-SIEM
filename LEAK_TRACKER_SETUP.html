This advanced DLP feature tracks how emails are forwarded from the original sender through multiple recipients, identifying when sensitive information reaches unauthorized destinations like competitors, personal emails, or public media.
1 Install Leak Tracking Tables

First, install the database tables for leak tracking:

  1. Open Command Prompt in: C:\xampp\htdocs\networkscan
  2. Run: php install_leak_tracking.php
  3. Wait for success message

This creates 4 new tables: email_forwarding_chains, email_recipients, leak_incidents, and domain_classifications.

2 Generate Sample Leak Chains

Create realistic sample data showing various leak scenarios:

  1. In the same command prompt, run: php generate_leak_chains.php
  2. This creates 10 email forwarding chains
  3. Includes scenarios: employee leaks, competitor leaks, media leaks, etc.
3 Access the Leak Tracker

View and analyze leak incidents:

Open Leak Tracker Dashboard

Or navigate to: http://localhost/networkscan/mailscan_leak_tracker.php
